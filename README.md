# 🌿 Medicinal Plant Identification using Vision Transformers

This project is about identifying medicinal plants from leaf images using a Vision Transformer (ViT) model. Given an input image, the model predicts the corresponding plant class from a set of 40 medicinal plants.

The main goal of this project was to explore transformer-based models for image classification and understand how they perform compared to traditional CNN approaches.

---

## 📌 Overview

- ~5945 images across 40 classes  
- Vision Transformer (ViT) for classification  
- Combination of TensorFlow (data handling) and PyTorch (training)  
- Validation accuracy: **99.24%**

---

## 📂 Dataset

The dataset is organized into folders where each folder represents a plant class.
